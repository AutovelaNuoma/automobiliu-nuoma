✅ TAI YRA PILNAI VEIKIANTI PLATFORMOS VERSIJA SU SLAPTAŽODŽIU

1. Įrašyk priklausomybes:
   npm install

2. Paleisk lokaliai:
   npm run dev
   ir atidaryk http://localhost:3000

3. Norėdamas pasiekti admin zoną:
   http://localhost:3000/admin
   Slaptažodis: autovelapadangos

4. Firebase konfiguracija yra utils/firebase.ts

✅ Paruošta deploy'ui į GitHub + Vercel
